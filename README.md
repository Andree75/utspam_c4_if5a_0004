**Pra-Syarat :**

    - Flutter SDK (>=3.38.1)
    - Dart 3.10.0
    - Android Emulator / iOS Simulator / Physical Device

**Instalasi :**

1. Clone repository

        git clone https://github.com/Andree75/utspam_c4_if5a_0004.git
        cd utspam_c4_if5a_0004

2. Install dependencies

        flutter pub get

3. Jalankan aplikasi

        flutter run
